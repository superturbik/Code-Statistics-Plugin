# Code Statistics Per File
Author: Kaparov Igor Sergeevich M3107


## Description
A simple extension for Visual Studio Code that analyzes the current file and outputs statistics:
- amount of functions,
- amount of comments,
- amount of empty lines.


## Features
- Quick file analysis.
- Convenient output of results.


## Usage
1. Open any file.
2. Press `Ctrl+Shift+P`.
3. Enter **Analyze current file (Code Stats)**.
